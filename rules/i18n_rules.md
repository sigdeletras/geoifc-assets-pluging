# Reglas de Internacionalizacion

## Alcance

La politica multiidioma aplica solo a:

* interfaz del complemento
* `metadata.txt`
* `README.md`
* `README.es.md`

La documentacion del repositorio se mantiene en espanol.

---

## Idiomas obligatorios

La interfaz debe estar disponible en:

* ingles
* espanol

---

## Textos traducibles

Deben ser traducibles:

* menus
* toolbar actions
* dialogs
* docks
* botones
* etiquetas
* mensajes de error
* mensajes de estado
* tooltips
* textos del visor embebido cuando sean UI del plugin

No introducir textos visibles hardcodeados.
