# Flujos y Historias de Usuario — GeoIFC Assets

| Flujo | Codigo HU | Titulo | Componente | Descripcion |
|-------|-----------|--------|------------|-------------|
| F01 | HU-10 | Instalar y validar el complemento desde el repositorio | Sistema | Como administrador del complemento, quiero una estructura de repositorio clara y empaquetable, para instalar el plugin en QGIS y mantenerlo en GitHub. |
| F01 | HU-20 | Instalar complemento desde ZIP | Sistema | Como usuario, quiero poder instalarme el complemento a partir del ZIP, para usarlo en mi instalacion de QGIS sin necesidad de acceso a un repositorio de complementos. |
| F02 | HU-01 | Validar la capa GIS de trabajo | Integracion GIS | Como tecnico GIS, quiero seleccionar una capa GIS que contenga ifc_path o ifc_url, para que el complemento pueda localizar el modelo IFC de cada feature de forma predecible. |
| F02 | HU-02 | Abrir el IFC asociado en un visor embebido desde feature seleccionado | Integracion GIS | Como gestor de activos, quiero abrir el IFC asociado al feature seleccionado dentro de QGIS, para revisar el modelo sin salir del entorno GIS. |
| F03 | HU-03 | Seleccionar un elemento IFC | Visor 3D | Como tecnico BIM, quiero seleccionar un elemento del modelo IFC, para consultar sus datos especificos y decidir que propiedades llevar a GIS. |
| F03 | HU-04 | Consultar propiedades BIM | Visor 3D | Como tecnico GIS, quiero consultar atributos, Property Sets y Quantity Sets del elemento IFC seleccionado, para entender que informacion esta disponible. |
| F03 | HU-05 | Seleccionar propiedades para cargar en GIS | Visor 3D | Como tecnico GIS, quiero marcar una o varias propiedades IFC, para preparar la informacion que se incorporara como atributos GIS. |
| F03 | HU-13 | Exportar CSV y JSON desde el visor IFC | Visor 3D | Como tecnico BIM, quiero exportar el arbol fisico o las propiedades del elemento IFC seleccionado a JSON o CSV, para documentar y compartir datos del modelo fuera de QGIS. |
| F03 | HU-14 | Navegacion en el visor 3D | Visor 3D | Como usuario del complemento, quiero navegar el modelo IFC en 3D con controles de orbita, zoom, pan y vistas predefinidas, para explorar su geometria desde diferentes angulos y escalas. |
| F04 | HU-15 | Consultar propiedades recopiladas | Propiedades | Como responsable de inventario, quiero consultar el listado de propiedades recopiladas del modelo IFC, para revisar que informacion esta disponible antes de transferirla a GIS. |
| F04 | HU-19 | Generar capa temporal GIS con propiedades | Propiedades | Como tecnico GIS, quiero crear una capa GIS temporal indicando su geometria y cargar las propiedades seleccionadas, para iniciar un inventario geoespacial con informacion BIM. |
| F05 | HU-16 | Transferir propiedades a GIS | Propiedades | Como responsable de inventario, quiero transferir las propiedades IFC seleccionadas al feature de la capa GIS, para enriquecer el inventario con informacion BIM. |
| F06 | HU-12 | Visualizar un IFC y crear o enriquecer una capa GIS sin capa previa | Integracion GIS | Como tecnico GIS que acaba de arrancar el complemento sin una capa GIS de activos, quiero abrir un IFC directamente y generar o enriquecer una capa GIS, para iniciar el flujo de inventario sin necesidad de preparar una capa previa. |
| F07 | HU-17 | Herramientas de analisis del visor | Visor 3D | Como tecnico BIM, quiero utilizar herramientas de analisis visual (medicion de distancia, area, seccion transversal y vistas ortonormales) en el visor 3D, para medir y explorar el modelo en detalle. |
| F08 | HU-09 | Usar el complemento en ingles y espanol | Integracion GIS | Como usuario del complemento, quiero poder usar toda la interfaz en ingles o espanol, para trabajar en el idioma requerido por mi entorno. |
| F08 | HU-18 | Ver los textos de propiedades en el idioma de QGIS | Integracion GIS | Como usuario del complemento, quiero ver los nombres y textos del panel de propiedades en el idioma configurado en QGIS, para trabajar en el idioma requerido por mi entorno. |
| F08 | HU-E04 | Generar huella geografica de una planta IFC georreferenciada como capa temporal | Integracion GIS | Como tecnico GIS, quiero seleccionar una planta de un modelo IFC georreferenciado y cargar su geometria 2D como capa temporal en QGIS, para verificar y situar geograficamente una planta concreta del edificio. |
| F09 | HU-11 | Consultar el estado de los flujos ejecutados | Sistema | Como usuario del complemento, quiero recibir mensajes claros sobre las operaciones realizadas, para saber si el flujo se ha completado correctamente o si debo corregir algo. |
