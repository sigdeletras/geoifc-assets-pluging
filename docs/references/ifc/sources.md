# Fuentes IFC-BIM

Este documento lista fuentes de referencia para los esquemas IFC.

---

# 1. Fuentes oficiales

Fuentes principales:

* buildingSMART IFC standards: https://www.buildingsmart.org/standards/bsi-standards/industry-foundation-classes/
* buildingSMART IFC schema specifications: https://technical.buildingsmart.org/standards/ifc/ifc-schema-specifications/
* IfcOpenShell documentation: https://docs.ifcopenshell.org/

---

# 2. Licencia y uso

Antes de incorporar esquemas, documentos o fragmentos oficiales al repositorio, revisar su licencia.

Reglas:

* guardar siempre la fuente original
* no modificar documentos oficiales sin dejarlo claro
* preferir enlaces y notas propias cuando haya dudas de licencia
* documentar fecha de consulta
* evitar copiar grandes bloques de documentacion externa

---

# 3. Artefactos recomendados

Cuando sea necesario, documentar para cada version:

* enlace oficial HTML
* EXPRESS schema si se decide versionarlo
* notas de compatibilidad
* entidades relevantes para el MVP
* Property Sets frecuentes
* Quantity Sets frecuentes
