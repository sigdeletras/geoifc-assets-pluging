# Referencias IFC-BIM

Esta carpeta contiene la base documental IFC-BIM versionada del proyecto GeoIFC Assets.

Su objetivo es servir como fuente de consulta para:

* lectura de modelos IFC
* interpretacion de propiedades
* mapeo IFC-GIS
* diferencias entre versiones IFC
* criterios de soporte del MVP

La documentacion de referencia vive dentro de `docs/references/ifc/`.

Los archivos IFC usados como datos de prueba deben vivir en `tests/fixtures/ifc/`.

---

# Versiones objetivo

Versiones a documentar inicialmente:

* IFC2x3 TC1
* IFC4 ADD2 TC1
* IFC4.3 ADD2 / IFC 4.3.2.0

---

# Documentos

* `versions.md`: matriz de versiones IFC soportadas.
* `sources.md`: fuentes oficiales y licencias.
* `properties_and_quantities.md`: Property Sets, Quantity Sets y lectura de propiedades.
* `ifc_gis_mapping.md`: criterios de mapeo IFC-GIS.
* `IFC2X3_TC1/README.md`: notas especificas de IFC2x3.
* `IFC4_ADD2_TC1/README.md`: notas especificas de IFC4.
* `IFC4X3_ADD2/README.md`: notas especificas de IFC4.3.
